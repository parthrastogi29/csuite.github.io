What is C-Suite

C-Suite has been made keeping in view the new trend of Large Language Models and A.I, though currently in beta stage,It can still be very usefull in automating various tasks.

Features
  • Excel-Hand module allows you to interact with your excel files with more efficiency. You can merge/demerge and manipulate data in multiple excel workbooks and sheets simultaneously.
  • Swayum module allows you to automate various computer commands
  • Audacity module allows you to download YouTube video, audio, or playlists
  • Chat-PDF module allows you to interact with your PDF files
  • Note: You will need to paste a free API-Key from https://www.chatpdf.com/docs/api/backend into the .env file inside the main program folder which will allow you to use ChatPDF backend API for free with 500 messages and 5,000 PDF pages per month. This should be enough for limited usage. If you require more usage, we encourage you to buy their API for more extensive usage. (It must be further noted that we are in no way associated with Chat-PDF but we love to use their service in our product and are thankful to them for developing the API which is used by C-Suite)
  • Parser module allows you to extract web pages and do web scraping to get data
  • Grepper module allows you to find text from one or more PDF files and paste it into a simple excel worksheet
  • Comparer module allows you to simply compare two or more excel files
  • A.N.A.N.T A large language module that can be installed and run locally(offline) without internet basically works like Chat-GPT. Coming Soon!
  • Others Many more modules are still in development and will soon be available via the next releases of C-Suite
